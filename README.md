# Personal Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my profile, education, skills, projects, and contact information.

## Sections Included
- Home
- About
- Education
- Skills
- Projects
- Contact

## Features
- Fully responsive design
- Smooth navigation between sections
- Dark/Light mode toggle
- Project showcase with GitHub and live demo links
- Contact section layout

## Technologies Used
- HTML
- CSS
- JavaScript
